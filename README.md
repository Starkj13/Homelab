<h1 align="center">Homelab</h1>

### Overview:
A basic bash script for easy installment of the most necessary things for a homlab.

Ubuntu based and in beta stage.

### Content
The script installs

✅Samba
✅Docker
✅Sonarr
✅Radarr
✅Jackett
✅Pingvinshare
✅Dashy
✅Nginx proxy manager 
✅Jellyfin

### Installation
For Ubuntu 
```sh
$ wget https://raw.githubusercontent.com/starkj13/main/bash.sh
$ chmod +x bash.sh
$ sudo ./bash.sh
```
